# 3xui Telegram Bot

Minimal Telegram bot + admin panel integration for 3xui.

## Minimum Requirements

- **CPU:** 1 vCPU
- **RAM:** 1 GB minimum (2 GB recommended)
- **Disk:** 5 GB free minimum (10 GB recommended)
- **OS:** Linux (Ubuntu 22.04+ recommended)
- **Network:** Public domain + open ports `80` and `443`

## Quick Start

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/03hitalin21/3xui-tgbot/New9/install.sh)
```

## Environment

Copy and edit the example environment file:

```bash
cp config/bot.env.example .env
```

Then run the installer and complete setup.
